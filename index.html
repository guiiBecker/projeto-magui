Sobre o Projeto

O Magui Web é um sistema de gestão desenvolvido especialmente para confecções, oferecendo funcionalidades completas para gerenciamento de pedidos, produtos, clientes e ordens de produção. O sistema foi projetado para ser intuitivo, eficiente e adaptado às necessidades específicas do setor têxtil.

Frontend Moderno

React, TypeScript e Tailwind CSS

Backend Robusto

FastAPI e PostgreSQL
